Welcome to log4js-ext!

Log4js-ext is a logging library that works with ExtJs.

If you are familiar with log4j or any other logging library based on log4j, then you already know log4js-ext.

Log4js-ext is Open Source, licensed under the GNU LGPL.

Main features

  • Log4j-like API: reuse your know-how!
  • Powerful built-in log viewer: with filtering, sorting, syntax highlighting and more...
  • Extensible: from simple message formatting to remote server-side logging
